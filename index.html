Takasbank Para Yatırma Rehberi

Takasbank bireysel bankacılık hizmeti sunmaz. Ancak sermaye piyasası işlemleri için para gönderme işlemlerinizde aracı kurumların Takasbank üzerindeki hesapları kullanılmaktadır.

Para Yatırma Noktası: Takasbank IBAN

Para gönderirken aşağıdaki bilgileri bilmeniz gerekir:

  • IBAN: Aracı kuruma özel Takasbank IBAN'ı
  • Açıklama: Hesap numaranız, ad-soyad veya referans kodu
🔔 Not: Açıklama kısmını eksiksiz doldurmazsanız işleminiz gecikebilir.
